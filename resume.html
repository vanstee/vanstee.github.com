Interests

While I'm still super happy writing web applications in Ruby, I've been pretty interested in distributed systems for a while, specifically Riak and Erlang/Elixir. Most recently I've been reading about LevelDB, ICE and STUN, CRDTs and Raft.

The sweet spot between my interests and experience would probably be hacking on highly available HTTP APIs, maybe with some kind of interesting storage piece. I've also been following some of the new JavaScript MVC frameworks lately, including Angular and Ember. So a chance to consume a nicely written API with one of those is definitely something I'm interested in as well.
